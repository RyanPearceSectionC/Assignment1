What exactly is C#?

C# is a programming language that was created with the intention and mind of making an easier and better version of C++. Unfortunately, due to it's restrictions, C# is slower than C++ but it's much easier to organize, manage and review code in my personal opinion.

C# is, in my opinion again, . It's classes have everything together so it's easy to read compared to C++ which has two files for one class. C# doesn't have as much access to creating programs or anything super advanced without using .NET or some sort of alternative programming language behind the scenes.

What Tools Do I Need?

Down Below, I am going to send links, along what their background. These are some compilers which will allow you to program in C#.

Visual Studio
Visual Studio is a tool that is used widely with good reason. It's quick to set-up, it has everything you could ask for, has intelli-sense to help you remember and put in methods/functions into your code fast. You really can't go wrong with it.
Visual Studio - Code
Visual Studio Code is good for smaller or more closed projects. It's basically just Visual Studio Lite. The only thing to note about this compared to Visual Studio is that the hierchy system is a little strange and you are limited away from .NET plugins and are instead forced to use their plugins if you wish to do something specific. Chances are, there is something that allows .NET support but until then, you're limited when using something like C#.
Sublime Text 3
Sublime Text 3 is a difficult but efficient and customizable text editor/compiler. You don't get the benefit to intell-sense like Visual Studio and Visual Studio Code does but in return, you have an easy to understand compiler that you are able to put the inputs in for yourself. If you can figure out what compiler you want to use and how you want to use it, this would be a good option. If not, I would say refer to the previous two.
Notepad++
Notepad++ is a text editor and shouldn't really be used for compiling but if you plan to write things like HTML or little scripts then it's definitely something to look into. It's basically if Visual Studio and Notepad fused together with both a community that will create 'intelli-sense'-like scripts that will help you on your programming/scripting journey. Unless you are certain that what you have is in there though, I would stick to Visual Studio and Visual Studio Code.
