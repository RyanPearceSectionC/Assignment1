What Tools Do Programmers Use/Need?

There are quite a few different tools that developers can use for Programming. There is never really 'set' tools that a developer must use but there are definitely a lot of options that'll help you out. The most important to me is the text editor as it can make your journey crystal clear or confusing.

Name Description Link
Visual Studio Visual Studio is a tool that is used widely with good reason. It's quick to set-up, it has everything you could ask for, has intelli-sense to help you remember and put in methods/functions into your code fast. You really can't go wrong with it. Click Here!
Visual Studio - Code Visual Studio Code is good for smaller or more closed projects. It's basically just Visual Studio Lite. The only thing to note about this compared to Visual Studio is that the hierchy system is a little strange and you are limited away from .NET plugins and are instead forced to use their plugins if you wish to do something specific. Chances are, there is something that allows .NET support but until then, you're limited when using something like C#. Click Here!
Sublime Text Sublime Text 3 is a difficult but efficient and customizable text editor/compiler. You don't get the benefit to intell-sense like Visual Studio and Visual Studio Code does but in return, you have an easy to understand compiler that you are able to put the inputs in for yourself. If you can figure out what compiler you want to use and how you want to use it, this would be a good option. If not, I would say refer to the previous two. Click Here!
Notepad++ Notepad++ is a text editor and shouldn't really be used for compiling but if you plan to write things like HTML or little scripts then it's definitely something to look into. It's basically if Visual Studio and Notepad fused together with both a community that will create 'intelli-sense'-like scripts that will help you on your programming/scripting journey. Unless you are certain that what you have is in there though, I would stick to Visual Studio and Visual Studio Code. Click Here!
